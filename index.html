Email service setup

Emails are sent via Resend (free tier — 100 emails/day).

Setup steps:

  1. Go to resend.com and create a free account
  2. Copy your API key from the dashboard
  3. In Vercel → Project Settings → Environment Variables
  4. Add variable: RESEND_API_KEY = your key
  5. Redeploy the site

Until the API key is added, notification emails will be silently skipped.
